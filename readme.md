---

Overview

This is a minimal, developer-style personal website built to present current work, projects, and ideas.

The site focuses on clarity and simplicity rather than heavy frameworks or complex tooling.

Current projects featured:

- DevPilot — AI coding agent that works with real repositories
- FollowUpKit — follow-up automation system for freelancers and consultants
- Creovate — digital products brand for productivity systems and workflow templates

---

Philosophy

The website intentionally follows a few principles:

- minimal design
- text-first content
- fast loading
- zero frameworks
- human-written HTML & CSS

It serves as a living log of projects and systems currently being built.

---

Tech Stack

The site is intentionally simple:

- HTML
- CSS
- minimal JavaScript
- deployed on Vercel

Fonts:

- Instrument Serif
- Geist Mono

---

Structure

index.html
style.css
assets/

- "index.html" — main page structure
- "style.css" — layout and styling
- "assets/" — images, icons, and metadata

---

SEO & GEO

The site includes:

- OpenGraph metadata
- Twitter cards
- structured schema.org data
- canonical URLs

These help both search engines and AI systems understand the identity and work of the founder.

---

Deployment

The site is deployed on Vercel and automatically updates when changes are pushed to the repository.
